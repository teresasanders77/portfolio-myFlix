Documentation

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

The table below describes each endpoint of the myFlix API and how to access it.

ENDPOINT URL DESCRIPTION METHOD PARAMETERS REQUEST RESPONSE
/movies This endpoint returns a list of all movies to the user. GET None None JSON object: Data about all of the movies.
/movies/:Title This endpoint retrieves data on a single movie (Genre, Director, Actors, Image URL) by movie title. GET None None JSON object: Data about a single movie.

Example:

{
"Genre": {
  "Name": "Mystery",
  "Description": "A mystery film is a genre of film that revolves around the solution of a problem or a crime. It focuses on the efforts of the detective, private investigator or amateur sleuth to solve the mysterious circumstances of an issue by means of clues, investigation, and clever deduction."
},
"Director": {
  "Name": "Alfred Hitchcock",
  "Bio": "An English film director and producer. He is one of the most influential and extensively studied filmmakers in the history of cinema.",
  "Birth": "1899",
  "Death": "1980"
},
"Actors": [],
"_id": "5f244290cb6336e9e2ef0281",
"Title": "Vertigo",
  "Description": "A former police detective juggles wrestling with his personal demons and becoming obsessed with a hauntingly beautiful woman.",
"ImagePath": "vertigo.jpg",
"Featured": false
}

/movies/genres/:Name This endpoint retrieves data on a certain genre by name. GET None None JSON object: Definition of a specific genre.

Example:

{ "Name": "Adventure",
  "Description": "Adventure films typically use their action scenes to display and explore exotic locations in an energetic way."
}

/movies/directors/:Name This endpoint provides information about a specific director (Bio, birth year, death year) by name. GET None None JSON object: Data about a director.

Example:

{
"Name": "Steven Spielberg",
  "Bio": "An American film director, producer, and screenwriter. He is considered one of the founding pioneers of the New Hollywood era and one of the most popular directors and producers in film history. Spielberg started in Hollywood directing television and several minor theatrical releases.",
  "Birth": "1946",
  "Death": "-"
}

/users This endpoint allows a new user to register (Username, password, email, date of birth). POST None JSON object: Data about the user to add.

Example:

{
"Username" : "johnjohn99",
"Password" : "johnjohnpassword",
"Email" : "johnjohn99@aol.com",
"Birthday" : "1955-05-05"
}

JSON object: Data about the user that was added, including an ID and any favorites that have been added to the users list.

Example:

{
"Favorites": [],
"_id": "5f29a2b080bcd00607219c7a",
"Username": "johnjohn99",
"Password": "johnjohnpassword",
"Email": "johnjohn99@aol.com",
"Birthday": "1955-05-05T00:00:00.000Z"
}

/users/:Username This endpoint allows current users to update their info (Username, password, email, date of birth). PUT None JSON Object: Data that is to be updated.

Example:

{
"Username" : "teresasanders77",
"Password" : "thisisatest",
"Email" : "teresasanders77@gmail.com",
"Birthday" : "1989-07-11"
}

JSON Object: Data (user, username, password, email, and date of birth) that was successfully updated. Example:

{
"Favorites": [ "5f244052cb6336e9e2ef027d" ],
"_id": "5f24448acb6336e9e2ef0282",
"Username": "teresasanders77",
"Password": "thisisatest",
"Email": "teresasanders77@gmail.com",
"Birthday": "1989-07-11T00:00:00.000Z"
}

/users/:Username/Movies/:MovieID This endpoint allows a user to add a specific movie to their list of favorites. POST None None JSON object: Movie ID that was successfully added to their list of favorites along with the rest of the user information.

Example:

{
"Favorites": [ "5f244052cb6336e9e2ef027d", "5f24413acb6336e9e2ef027f", "5f243a07cb6336e9e2ef0277" ],
"_id": "5f244566cb6336e9e2ef0285",
"Username": "janedoe01",
"Password": "janedoesdog",
"Email": "janedoe01@gmail.com",
"Birthday": "1999-09-19T00:00:00.000Z"
}

/users/:Username/Movies/:MovieID This endpoint allows a user to remove a movie from their list of favorites. DELETE None None JSON Object: Updated list of favorites showing the successful removal of the specific movie, as well as the rest of the information of the user.

Example:

{
"Favorites": [ "5f244052cb6336e9e2ef027d", "5f243a07cb6336e9e2ef0277" ],
"_id": "5f244566cb6336e9e2ef0285",
"Username": "janedoe01",
"Password": "janedoesdog",
"Email": "janedoe01@gmail.com",
"Birthday": "1999-09-19T00:00:00.000Z"
}

/users/:Username This endpoint allows a current user to de-register their information. DELETE None None A text message indicating whether the user was successfully deregistered.

Example:

testuser1003 was deleted.
